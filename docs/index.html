Android

Use Moneytiser to power your Android projects

Create Android-based applications and use the Android SDK to create a tunnel with Moneytise servers. The SDK uses Moneytise APIs.

Getting Started

This guide will help you get started with our Android SDK to build apps powered by Moneytiser.

Prerequisites

To get started with Android SDK, you will need one of the following: Android Studio or Eclipse.

SDK Installation and Setup

To add the Moneytiser Android SDK to your existing project, perform the steps given below:

  1. Download the Android SDK and extract the ZIP file to your local disk.
  2. Add references/dependencies using Android Studio:

    3. Android Studio:

    1. Copy the 'moneytiser-sdk-x.x.x.aar' file into your project's libs folder.
    2. Add the dependency code into your 'build.gradle' file.
      3. compile fileTree(dir: 'libs', include: ['*.jar'])
    3. Configure 'AndroidManifest.xml' with permissions using the following code:
      4. <!-- Allows applications to connect network (Required) -->
<uses-permission android:name="android.permission.INTERNET" />
<!-- Allows applications to access information about networks (Required) -->
<uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />

Initialize SDK

To initialize the SDK, specify application context and the Publisher key of the stack as shown in the snippet given below: 

Moneytiser m = new Moneytiser.Builder().withPublisher("publisher").build(context);

Once you have initialized the SDK, you can start and stop proxy connection.

API Reference

Go through our SDK API Reference guide to know about the methods that can be used to start and stop proxy connection.
